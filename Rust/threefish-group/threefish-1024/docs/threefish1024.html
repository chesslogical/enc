A lightweight, secure command-line tool for encrypting and decrypting files using the Threefish1024 cipher with HMAC-SHA256 authentication.

üöÄ Features

  • Uses Threefish-1024 block cipher as a stream cipher
  • Encrypts and decrypts files in-place
  • Verifies file integrity and authenticity with HMAC-SHA256
  • Secure key handling using zeroize
  • Cross-platform, no unsafe code

üß™ Requirements

  • A key file named key.key in the same directory as the binary
  • Key must be exactly 160 bytes
160 bytes total:
- First 128 bytes: Encryption key (Threefish)
- Last 32 bytes:   HMAC key (SHA256)
Note: You can generate a secure 160-byte key using a separate keygen tool or /dev/urandom.

üîß Usage

Encrypt a file

threefish_cli encrypt myfile.txt

Decrypt a file

threefish_cli decrypt myfile.txt

The tool encrypts or decrypts the file in-place. A temporary file is created and then replaces the original after success.

  • HMAC-SHA256 ensures no tampering goes undetected
